# DevOps Website

DevOps Website is a responsive multi-page web application developed using HTML5 and CSS3. The project was created as part of a Web Development Internship Task to demonstrate the fundamentals of front-end web development, responsive design, and website structure.

## Features

* Responsive and mobile-friendly design
* Home, About, Services, and Contact pages
* Consistent navigation bar and footer across all pages
* DevOps-themed image gallery
* Contact form for user inquiries
* Google Maps integration for location display
* Clean and modern user interface

## Technologies Used

* HTML5
* CSS3
* Git
* GitHub
* Responsive Web Design
